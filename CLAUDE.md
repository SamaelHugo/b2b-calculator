# B2B Calculator Landing Page

## Project
Single-page B2B landing with step-by-step quiz-calculator.
Demo project for web developer portfolio.

## Technical Rules
- Single index.html file, all CSS and JS inline
- Vanilla HTML/CSS/JS, no frameworks
- Do NOT change CSS variables after Prompt 0
- Do NOT change fonts after Prompt 0
- Russian language for all text
- Mobile responsive (768px, 1024px)

## Design Direction
- Dark B2B theme, kontur.ru/kedo aesthetic
- One accent color only
- No purple gradients, no Inter/Roboto/Arial
- Premium, "expensive" feel
